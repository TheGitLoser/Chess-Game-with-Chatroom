# Tools used
Server-side
> Node.js, Express.js, Socket.io, Jade(Template engine)

Client-side
> Bootstrap 3, jQuery, HTML, JavaScript

# Getting Started

1. Download
```
git clone https://github.com/TheHKCoder/Chess-Game-with-Chatroom.git
```

2. Install
```
cd Chess-Game-with-Chatroom
npm install
```

3. Run
```
node server.js
```

4. URL
```
http://localhost:3000/
```
